- INFORMACIÓN DEL PROYECTO
Nombre del Proyecto: LinkSnap - URL Shortener API
Se trata de: Es una API asíncrona de alto rendimiento para acortar enlaces, rastrear visitas y obtener estadisticas.
Tencologías: Python 3.10+, FastAPI, SQLModel (SQLAlchemy + Pydantic), SQLite Async (aiosqlite)

- COMANDOS PRINCIPALES PARA EJECUTARLO
Instalación de dependencias: pip install -r requirements.txt
Ejecutar el servidor: uvicorn main:app --reload
